== Description ==

**Third-Party Service Disclosure**

This plugin relies on a third-party service, Vocalize.africa, to convert text into audio. When you create, update, or delete a post, data such as the post's title, content, author name, publication date, and featured image is sent to Vocalize's API to enable the conversion of the posted Article into Audio.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/text-to-audio-with-ads` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings->Text to Audio Settings screen to configure your API Keys. 

== Frequently Asked Questions ==

1. How do I create API keys for use with this plugin?

Visit wwww.vocalize.africa, register as a Publisher to generate your API keys 

2. Does this plugin work with custom post types?
Yes, this plugin works with all post types, including custom post types.

3. How does the plugin convert text to audio? 
The plugin sends post data to the Vocalize service, which processes the content and generates an audio file. The audio player is then embedded in the post.
